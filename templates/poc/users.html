Users API POC

This lightweight dashboard helps you exercise the /api/users/ endpoints.

Workflow suggestion:

  1. Register a user (optional if you already have credentials).
  2. Authenticate to retrieve JWT tokens.
  3. Call profile/roles/entitlements endpoints.
  4. Refresh the access token when it expires.
